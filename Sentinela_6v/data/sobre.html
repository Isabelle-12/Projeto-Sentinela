Changelog do Sistema (Histórico de Versões)
v5.0.0 11/05/2026
  • Correção da distribuição Dual-Core: tasks críticas (Sensors, Security) movidas para Core 0.
  • Adição de handles em todas as tasks para controle em tempo de execução.
  • Finalização do JavaScript de todas as páginas da interface web.
v4.0.0 11/05/2026
  • Migração do banco de dados de MySQL externo para SQLite local via LittleFS.
  • Correção do Watchdog Timer para ESP32 SDK 3.x (esp_task_wdt_config_t).
  • Implementação de filtro anti-queda elétrica no sensor MQ-2.
  • Debounce de software (2 leituras consecutivas) para evitar falsos positivos.
  • Deadzone de 2s após desligamento do alarme para evitar repique elétrico.
  • Acionamento escalonado (500ms entre buzzer e relé) para evitar brownout.
  • Variáveis volatile para sincronização em tempo real entre tasks.
  • Alteração do Partition Scheme para Huge APP (3MB).
v3.0.0 07/05/2026
  • Implementação de banco de dados MySQL com taskMySQL no Core 0.
  • Adição de Watchdog Timer (WDT) com timeout de 5 segundos.
  • Comunicação entre tasks via fila (QueueHandle_t).
  • Ajuste de prioridades: Security(P5), Sensors(P4), Web(P3), Perf(P2), MySQL(P1).
  • Novos endpoints de API: /api/status e /api/performance.
v2.0.0 06/05/2026
  • Integração completa do SPIFFS com servidor de arquivos e MIME types.
  • API JSON em /api/data com gasPPM, distancia e alarme.
  • Leitura funcional do HC-SR04 via pulseIn.
  • Retenção de segurança de 10s (MIN_TIME_ACTIVE) no alarme.
  • Filtro de valores espúrios (>4000) no ADC do MQ-2.
  • Divisor de tensão resistivo para proteção do GPIO 34.
  • Interface web com 5 páginas: Dashboard, Performance, Config, Logs e Sobre.
v1.0.0 29/04/2026
  • Estrutura base do firmware com 5 tasks FreeRTOS.
  • Leitura do sensor MQ-2 e acionamento de buzzer, relé e servo.
  • Configuração do Wi-Fi em modo Access Point (Sentinela_ESP32).
  • Servidor web básico com endpoint raiz.
