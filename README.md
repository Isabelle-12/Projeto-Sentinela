# Projeto-Sentinela
O Projeto Sentinela desenvolve um sistema ciberfísico com ESP32 capaz de detectar vazamentos de gases inflamáveis e princípios de incêndio. O sistema permite monitoramento e controle remoto por uma interface web embarcada e conta com mecanismos de segurança física e lógica, incluindo autenticação por RFID para controle manual.
